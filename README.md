# Fake Blade

# ゲーム概要
* タイミングで敵を倒す、スマホ向けアクションゲーム

# 操作方法
* 攻撃：スワイプ
* パリィ：中央長押し
* 回避：左右ボタン

# 工夫したポイント
* 攻撃のアニメーションに合わせたヒットエフェクトの生成
* 操作方法を直感的にわかりやすくするために操作の種類を最小限に抑えた

# 使用アセット

* PolyPeople Series - MedievalEurope: Peasants [Free]
* Low Poly Atmospheric Locations Pack
* Low Poly RPG Fantasy Weapons Lite
* Stylized Low Poly Skeleton
* Skybox Series Free
