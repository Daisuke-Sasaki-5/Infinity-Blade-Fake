# Fake Blade

# ゲーム概要
* タイミングよくカードや回避を繰り返して敵を倒す、スマホ向けアクションゲーム

# ルール
* ⚠マークがでた場所と逆の場所に回避することで回避成功、攻撃が通る
* 真ん中の場合、タイミングよくパリィすることでパリィ成功、攻撃が通る
* どちらかのHPが0になるまで続く

# 操作方法
* 攻撃：スワイプ
* パリィ：中央長押し
* 回避：左右ボタン

# 工夫したポイント
* 攻撃のアニメーションに合わせたヒットエフェクトの生成
* 操作方法を直感的にわかりやすくするために操作の種類を最小限に抑えた
